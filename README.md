
# Lost & Found App - Pass Task 7.1

An Android mobile application built with Java and SQLite that helps connect lost items to their owners.

## Features

- Post lost or found item adverts with image, date, location and category

- View all listings with Lost/Found filter buttons

- SQLite local database storage

- Date/time stamp on every posting

- Remove advert once owner is found

- Filter by category (Electronics, Pets, Wallets, Keys, Bags, Clothing)

- Image upload for each post

## Tech Stack

- Java

- Android SDK

- SQLite Database

- Android Studio

## How to Run

1. Clone the repository

2. Open in Android Studio

3. Run on emulator or physical device (API 24+)

## Screens

- Home Screen - Create advert or view all listings

- Create Advert - Fill in details and upload image

- Item List - View all posts with Lost/Found filter

- Item Detail - View full details and remove advert
